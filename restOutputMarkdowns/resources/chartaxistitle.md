# ChartAxisTitle

Represents the title of a chart axis.

## Properties
| Property	   | Type	|Description|
|:---------------|:--------|:----------|
|text|string|Represents the axis title.|
|visible|bool|A boolean that specifies the visibility of an axis title.|

## Relationships
| Relationship | Type	|Description|
